Foundry VTT Installer

Instructions

Option 1: Download using URL

  1. Log into Foundry VTT at https://foundryvtt.com/
  2. Click your profile in the upper right hand corner
  3. Click purchased content
  4. Change "Download Version" to desired version
  5. Change "Operating System" to linux/nodejs
  6. Click time URL
  7. Paste it into the input field below

Option 2: Upload ZIP file directly

  1. Select the Foundry VTT ZIP file from your device
  2. Or drag and drop the file into the upload area
